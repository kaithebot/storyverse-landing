# New Era Furniture Product Extraction Plan

## Objectives
- Extract 150+ products from newerafurniture.us
- Cover all categories: sofas, sectionals, bedroom, dining, chairs
- Collect complete specifications, dimensions, features
- Organize data for paginated website (20 products per page)

## Categories to Target
1. **Sofas** - All sofa types and styles
2. **Sectionals** - Modular and sectional sofas
3. **Bedroom** - Beds, nightstands, dressers
4. **Dining** - Tables, chairs, sets
5. **Chairs** - Accent chairs, recliners, office

## Data Points to Extract
- Product name
- SKU/Model number
- Price (regular/sale)
- Dimensions (L×W×H)
- Weight
- Materials/Fabrics
- Colors/Finishes available
- Features and specifications
- Product description
- Images (URLs)
- Category classification
- Stock status

## Technical Approach
- Use Bright Data API for systematic crawling
- Handle pagination and category navigation
- Extract from product detail pages
- Validate data quality and completeness
- Structure output for web deployment

## Output Format
- JSON structure ready for web integration
- Organized by categories
- Paginated groups of 20 products
- Include metadata for web deployment
